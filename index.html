Premier League Power Index

FIXTURE GUIDE AND MATCH PREDICTIONS
FOR ALL PREMIER LEAGUE TEAMS

Use the charts below to find out who has the easiest or hardest set of matches based on the Premier League Power Index (PPI). The PPI is inspired by FiveThirtyEight's Soccer Power Index (SPI) and follows a very similar methology that computes Team Strength Ratings based on past performances using xG and Adjsuted Goals. Each team has an offense and defense rating which measure the expected goals scored/conceded against an average team in the leauge on neutral ground. Combined, these ratings create the Power Index (PPI).

Notes:
Team Strength Ratings (OFF | DEF) are derived from historical matches using shot-based xG (fbref) and Adjusted Goals scored (i.e., goals are downweighted when the opponent had an on-field disadvantage such as red cards and goals late in a match when already leading).
OFF: Offensive Team Strength representing the avg. number of goals is expected to score against the average team in the league on neutral ground (higher is better). DEF: Defensive Team Strength representing the avg. number of goals is expected to concede against the average team in the league on neutral ground (lower is better).

Match Predictions are then made using Team Strength Ratings using a Poisson model. See individual match predictions below.
Match Forecasts

For each upcoming match, we calculate the expected number of goals each team would score based on their Offense & Defense ratings, a home advantage and a league strength factor.

We then simulate match scores by using a Poisson distribution to get the full probability distribution for 0, 1, 2, 3, and 4+ goals for each team.
